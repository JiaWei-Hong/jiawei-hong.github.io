使用array_filter來過濾空數組

發表於2020-12-03|更新於2020-12-17|Note

Introduction

曾經在寫Laravel某個專案中，當時collect還不太了解，所以使用array_filter來過濾空數組。

arary_filter

array_filter(array $array [, callable $callback [, int $flag = 0 ]] ) : array

Result

Array (
[1] => A
[3] => B
[5] => C
)

Conclusion

從上方結果來看如果array_filter不丟callback的話，會濾掉你的陣列中等於False的元素。
